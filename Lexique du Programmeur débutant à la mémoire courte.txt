LEXIQUE DU PROGRAMMEUR DEBUTANT À LA MEMOIRE COURTE


Algorithme : Un ensemble ordonné d'étapes transformant un certain Input en un Output.

API : En informatique, une interface de programmation (souvent désignée par le terme API pour Application Programming Interface)
est un ensemble normalisé de classes, de méthodes ou de fonctions qui sert de façade par laquelle un logiciel offre des services
à d'autres logiciels. Elle est offerte par une bibliothèque logicielle ou un service web, le plus souvent accompagnée d'une description
qui spécifie comment des programmes consommateurs peuvent se servir des fonctionnalités du programme fournisseur.

Problème NP : (NP = Non-deterministic Polynomial time) Problème dont il est possible de trouver une solution en temps polynomial (2^(O(log(n))))
en utilisant une machine de Turing non-déterministe.

Structure de données (Data Structure) : Une structure de données est un moyen de stocker et d'organiser des données pour faciliter leur accès et leur modification.
